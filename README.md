# Projeto para poder fazer o Encode/ Decode de arquivos em excel para base64

# Como rodar o projeto

rodar npm i ou yarn para instalar as dependencias 


# ENCODE
colar o arquivo xlsx na raiz do projeto e no encode.ts referenciar o arquivo
e rodar npm run encode ou yarn encode

# decode

preencher o base64.txt com sua string em base64 e rodar npm run decode ou yarn decode
